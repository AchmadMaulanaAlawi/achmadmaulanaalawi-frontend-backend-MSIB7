Diagonal Sum

Given a square matrix. Calculate the sum of the elements on its diagonals.
The sum should only count the elements on the diagonals and there should be no duplication when summing.


Example:

[ [1, 2, 3], [4, 5, 6], [7, 8, 9], ]

Output:

25
